# SOLID — Guía de referencia

Recurso interactivo para aprender los principios SOLID de diseño orientado a objetos. Explicaciones formales, en palabras simples, ejemplos cotidianos y código real en Python y TypeScript.

**[Ver en vivo →](https://solid-principles-pages.bcastrolab.org/)**

## Contenido

Cada principio incluye:

- **Definición formal** — el enunciado canónico del principio
- **En palabras simples** — explicación sin jerga técnica
- **Ejemplo cotidiano** — analogía del mundo real con ilustración
- **Código** — lado a lado: la versión que viola el principio y la que lo respeta (Python + TypeScript)
- **Ejercicios** — botón para pedir un ejercicio de práctica sobre cada principio

## Navegación

- **Sidebar** con los 5 principios, resaltando tu posición actual
- **Breadcrumb + dots de progreso** en cada vista para saber exactamente dónde estás
- **Mini-mapa flotante** (tecla `M`) para saltar entre principios
- **Atajos de teclado**: `←` `→` para moverte entre secciones

## Tecnologías

HTML, CSS y JavaScript vanilla. Sin frameworks ni dependencias de build. Solo fuentes de Google Fonts.
